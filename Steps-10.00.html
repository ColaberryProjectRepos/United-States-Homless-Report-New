 Adding Additional Tables and measures 


 
In this step you are going to create additional tables and measures. This helps you organize the data and make your visualization easier. 
 
  • Create KPI Group table
 
 
 
 
  1.    Click on Transform Data
  2.    Copy and paste the smallest table by right clicking on the table title and press copy, then    press CTRL+V to paste
  3.    Rename the table to best fitting name to the data in the table 
  4.    Delete all the columns on the table
  5.    Insert an Index column by clicking Add Column and select Index Column From 1
  6.    Insert a Conditional Column to add the KPI YOY% Group
  7.   Create a measure to the new KPI table  to show data based on this new KPI Group. In this   measure, you will type the following DAX code: 
 
KPI = (SWITCH(SELECTEDVALUE('KPI Flow'[Index]),1,SUM('Total US Data'[Homeless YOY%]),2,sum('Total US Data'[M.Income YOY%]),3,sum('Total US Data'[Population YOY%]),4,SUM('Total US Data'[  US Inflation Rate ])))


 

  
  • Create Age Group Table
 

 
 
 
  1.   Click Transform Data
  2.   Copy and paste the KPI Data Flow table
  3.    Delete the KPI group
  4.    Add a new Conditional Column for the age group
  5.    Create 2 measures using the following DAX codes: 
 
Homeless by Age Group = (SWITCH(SELECTEDVALUE('Group By Age'[ID]),1,(sum('US Data By State'[Overall Homeless - Under 18])/DISTINCTCOUNT('US Data By State'[YEAR])),2,(sum('US Data By State'[Overall Homeless - Age 18 to 24])/DISTINCTCOUNT('US Data By State'[YEAR])),3,(sum('US Data By State'[Overall Homeless - Over 24])/DISTINCTCOUNT('US Data By State'[YEAR]))))

YOY% Total Homeless by Age = (SWITCH(SELECTEDVALUE('Group By Age'[ID]),1,CALCULATE(AVERAGE('US Data By State'[YOY%  Homeless Under 18]),'US Data By State'[YEAR ID] <> 1),2,CALCULATE(AVERAGE('US Data By State'[YOY%  18-24]),'US Data By State'[YEAR ID] <> 1),3,CALCULATE(AVERAGE('US Data By State'[YOY%  Over 24]),'US Data By State'[YEAR ID] <> 1)))
 


 
 
 
  •   Create Race Group Table



 
 
  1.   Follow the same steps as above to add Race Groups in the Conditional Column
  2.   Add measures to the Age Group and Race Group tables using the following DAX codes
 
Homeless by Age Group = (SWITCH(SELECTEDVALUE('Group By Age'[ID]),1,(sum('US Data By State'[Overall Homeless - Under 18])/DISTINCTCOUNT('US Data By State'[YEAR])),2,(sum('US Data By State'[Overall Homeless - Age 18 to 24])/DISTINCTCOUNT('US Data By State'[YEAR])),3,(sum('US Data By State'[Overall Homeless - Over 24])/DISTINCTCOUNT('US Data By State'[YEAR]))))
 
 YOY% Total Homeless by Age = (SWITCH(SELECTEDVALUE('Group By Age'[ID]),1,CALCULATE(AVERAGE('US Data By State'[YOY%  Homeless Under 18]),'US Data By State'[YEAR ID] <> 1),2,CALCULATE(AVERAGE('US Data By State'[YOY%  18-24]),'US Data By State'[YEAR ID] <> 1),3,CALCULATE(AVERAGE('US Data By State'[YOY%  Over 24]),'US Data By State'[YEAR ID] <> 1)))
