Creating Map (the first chart) 13
 In this step we will build a map to show the Data by State
 

 
Steps
 
  1.   In the Report View, open the visualization pane, then Select Filled Map from visualization
  2.   In the fields, Location = State, Color Saturation = Average No. Of Homeless, Tooltips :    Homeless YOY%, Income YOY% and Population YOY%
  3.   In the Formatting Tab- Visual-  Fill Colors ->Minimum =#D5D0F2 & Maximum= #201DE7,   General-->Properties-->Size: Height = 316, Width = 379, Position: Horizontal = 901, Vertical = 93, Title: State, Font = Segoe UI Bold, Size =13, Text Color= #ffffff, Background Color = #201DE7, Horizontal alignment= Center, Effects: Color= #fffffff, Transparency= 27%,Visual Border =ON, Color=#808080
