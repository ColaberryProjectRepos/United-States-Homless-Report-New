Importing the Data 8





    1. First you need to have the data save on your device, then open Power BI Desktop
    2.   Click Get data
    3.   Select Excel Workbook
    4.   Select and open Homeless Data From 2015 to 2019.xlsx file
    5.   Under navigator select Us data by state(this is the US data by state from 2015 to 2019) and select Load Data
    6.   Repeat the above last steps to load the Consolidated Source (Total   US).xlsx file ( this is   overall US Data from 2015 to 2019) to Power BI
    7.   Save the power BI project with a meaningful name
