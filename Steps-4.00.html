Problems Encountered


  1.  The data I choose for the project was sourced from three different data sources ( all have large data). When I load them to the Power BI desktop directly and try to create a relationship between the data sets or create measures, it slowed down my process. Therefore, I solved that problem by doing most of the combining and clean up of the data using excel before I upload them in to the power BI desktop.
  2. I was able to get data for the US Homeless, median income,and Population by state, which helped me to build the visual using State and region. However, there was no reliable data available for the inflation rate by state. Therefore, inorder to include inflation rate in my analysis and visual, I had to have two categories of data sets. The first one consists of  homeless, income, and population by state. The  second one consists of  total US homeless, median income, population and inflation.
