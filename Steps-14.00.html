Waterfall Chart 14
This is where we create a waterflow chart


 

  1.  Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart.
  2.   Drag the KPI Group to Category and KPI YOY to Y-axis fields respectively. Tooltips: Homeless YOY%, Income YOY%, Population YOY% and Inflation YOY%.
  3.   In the Formatting Tab-->Visual-->Callout value: Font = Segoe UI Bold, size = 28, Category label: Off, Color: #a66999, In the Formatting Tab-->Visual- X-axis=On, Y-axis=On, Legend=On, Columns : Increase =#3bb44a, Decrease = E81123, Show total column=Off, General-->Properties-->Size: Height = 312, Width = 383, Position: Horizontal = 163, Vertical = 408, Title: YOY, Font = Segoe UI Bold, Size = 13, Text Color = #FFFFF, Background Color= #3370C6, Center, Effect: Background: On, Color = #FAE996, Transparency = 26%,Visual border : On, Color=#808080,Rounded Corners =2p (follow the ff GIF)
