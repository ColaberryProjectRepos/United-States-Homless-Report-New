 Creating KPI Cards


                                     

 

 
Before you begin working on your KPI cards, Insert the following additional measures to the “Total US Data”  table (Refer to Step 3 GIF on how to insert a measure). This are created for the KPI Cards.


                                   


                           

 
 
 
ARROW Homeless YOY% = SWITCH(TRUE(),AVERAGE('US Data By State'[YOY% Total Homeless])<0,UNICHAR(9662),SWITCH(TRUE(),AVERAGE('US Data By State'[YOY% Total Homeless])=0,UNICHAR(9666),UNICHAR(9652))  )
 
ARROW Income YOY% = SWITCH(TRUE(),AVERAGE('US Data By State'[Income YOY%])<0,UNICHAR(9662),SWITCH(TRUE(),AVERAGE('US Data By State'[Income YOY%])=0,UNICHAR(9666),UNICHAR(9652))  )
 
ARROW Inflation % = SWITCH(TRUE(),AVERAGE('Total US Data'[  US Inflation Rate ])<0,UNICHAR(9662),SWITCH(TRUE(),AVERAGE('Total US Data'[  US Inflation Rate ])=0,UNICHAR(9666),UNICHAR(9652))  )
 
ARROW Popn. YOY% = SWITCH(TRUE(),AVERAGE('US Data By State'[Population YOY%])<0,UNICHAR(9662),SWITCH(TRUE(),AVERAGE('US Data By State'[Population YOY%])=0,UNICHAR(9666),UNICHAR(9652))  )
 
 
 
 A.  Create Homeless YOY% KPI Card-


                              

 

Main Card
 
1.    Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
2.    Drag the Average Homeless From US Data By State table to the fields value box.
3.    In the Formatting Tab-->Visual-->Callout value: Font = Segoe UI Bold, size = 28, Category label:          Off, Color: #a66999
4.    In the Formatting Tab-->General-->Properties-->Size: Height = 136, Width = 162, Position: Horizontal = 0, Vertical = 93, Title: Homeless | YOY%, Font = Segoe UI Bold, Size = 11, Text Color = #FFFFF, Background Color= #3370C6, Center, Effect: Background: On, Color = #FAE996, Transparency = 26%,Visual border : On, Color=#808080,Rounded Corners =2p (follow the ff GIF)
 


5.    For the Color changing arrow and for the % indicators Click a new Card and format the following formats:




                                                                                   
 Arrow card :

1.    Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
2.    Drag the ARROW Homeless YOY% to the fields value box.
3.    In the Formatting Tab-->Visual-->Callout value: Font = DIN  Bold, size = 38, Category label: Off, Color: use conditional formatting and adjust like you see in the screenshot below.
4.    In the Formatting Tab-->General-->Properties-->Size: Height = 54, Width = 83, Position: Horizontal = 43, Vertical = 172, Title: Off


               



 
 % Indicator card :

1.    Click on an empty space on the canvas, and from the visualizations pane select Visual Card.
