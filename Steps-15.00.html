Line and Stacked Column Charts 15

Year Chart



 
This chart is to show the data by year
 

  1.  Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart.
  2.  Drag the Year  to the X-axis, Average No. of Homeless to Column y-axis, Homeless YOY%, Income YOY%, Population YOY% and Inflation % to Line y-axis and to Tooltips (rename the visuals to fit)
  3.   In the Formatting Tab-->Visual- X-axis=On% Y-axis=On, Font = Segoe UI &10,Legend=On, Font = Segoe UI &8   
       General-->Properties-->Size: Height = 316, Width = 390, Position: Horizontal = 162, Vertical = 93, Title: YEAR, Font = Segoe UI Bold, Size = 13, Text Color = #FFFFF, Background Color=   #3370C6, Center, Effect: Background: On, Color = #FAE996, Transparency = 27%,Visual border : On, Color=#808080,Rounded Corners =2p Column: Color=#7d4f73, Lines: Style=Solid, Type=Round, Stroke=4px,Colors=#EA3037,7F312F,#2419EC
