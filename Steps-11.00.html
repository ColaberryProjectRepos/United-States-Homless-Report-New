Inserting Title and logo, Refresh Date and Slicers 11

 Insert Title and Logo

 
  
 
 
 
  1.  In the report view, Select Insert ribbon>Text box Type the Project Name ‘United States Homeless Report (2015 -   2019). Highlight the Text and adjust Size = 28, Font = Segoe (Bold), Center. In the Formatting Tab-->General-->Properties-->Size: Height= 94,Width = 846, Position: Horizontal = 133, Vertical = 0, Effect: Background: On, Color = #FFFFFF,Transparency =0% 
  2.   In the Report View, Select Insert ribbon > Image > and select the Colaberry logo.  In the Formatting Tab-->General--> Properties-->Size: Height =   36, Width = 238, Position: Horizontal = 103, Vertical =65,Background: Off. 
  3.   Repeat the same step 2 for the project Logo In the Formatting Tab-->General--> Properties-->Size: Height = 94, Width =133, Position: Horizontal = 0, Vertical = 0, Background: Off.  
  4. Follow the following Gif for step-by-step 


 
 
 

 

Creating  Refresh Date

 
                                                  
 
 
 
1.  In the Report View, open the visualization pane, then Select Card from visualization

2.  In the fields, open ‘US Data By State’ Table, drag the Refresh Date.

    a. In the Formatting Tab-->Visual-->Callout Value : Font = DIN & Bold, Size =10, Color = #1a1a1a
    b. In the Formatting Tab-->General-->Properties-->Size: Height = 84, Width = 162, Position: Horizontal = 0, Vertical = 636, Title: Refresh Date, Font = Segoe UI Bold, Size =10, Text Color= #ffffff, Background Color = #3370C6, Horizontal alignment= Center, Effects: Color= #FAE99F, Transparency= 26%,Visual Border =ON, Color=#808080,Rounded Corners=2 px
  Follow the following Gif for step-by-step instruction.



 
 

 Creating Year Slicer


                

 
 
 
1.  In the Report View, open the visualization pane, then Select Slicer from visualization
2.  In the fields, open ‘Total US Data’ Table, drag the Year.
    a. In the Formatting Tab-->Visual-->Values: Font = Segoe UI Bold, Size = 9, Slider: On
    b. In the Formatting Tab-->General-->Properties-->Size: Height = 94, Width =300,            Position: Horizontal = 980, Vertical = 0, Title: Year, Font = Segoe UI Bold, Size =11, Horizontal alignment= Center, Text Color= #ffffff,Background Color=#3370C6
